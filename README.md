# DermaCare

DermaCare is a modern skincare treatment booking app designed to simplify service discovery and appointment scheduling through a clean and user-centered mobile experience.

## Features
- Service-centered booking flow
- Treatment categories
- Booking progress indicator
- Upcoming appointment reminders
- Clear pricing and duration
- Minimal and modern UI

## User Flow
Login/Register
→ Browse Treatments
→ Treatment Details
→ Select Date & Time
→ Personal Details
→ Review Booking
→ Booking Success

## Tools Used
- Figma

## Design Focus
The project focused on improving usability, simplifying booking interactions, and creating a calm wellness-inspired mobile experience.
