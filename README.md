# Skinderm Dermatology App

## Overview

This project is a mobile appointment booking application designed for Skinderm Dermatology & Aesthetics, a skincare and dermatology clinic offering medical and aesthetic treatments.

The app was created to improve the clinic’s current booking process, which mainly relies on phone calls and WhatsApp communication.

---

## Problem Statement

Manual booking systems can lead to:

* missed appointments
* scheduling confusion
* delayed responses
* poor customer experience

This project provides a mobile-first solution that simplifies appointment scheduling for both patients and clinic staff.

---

## Project Goals

* Create a smooth mobile booking experience
* Reduce booking friction
* Improve appointment organization
* Design an accessible and user-friendly interface
* Build a professional medical aesthetic

---

## Features

* Login
* Treatment Categories
* Treatment Details
* Date & Time Slot Selection
* Appointment Confirmation
* Mobile-Friendly Navigation

---

## Target Users

* Patients seeking skincare treatments
* First-time consultation clients
* Returning patients booking follow-up appointments
* Busy users preferring quick mobile booking

---

## Design Process

The design process included:

1. Researching dermatology clinic booking systems
2. Identifying common booking problems
3. Planning the appointment booking flow
4. Creating mobile-first wireframes
5. Designing high-fidelity UI screens in Figma

---

## UX Decisions

* Simple navigation structure
* Touch-friendly buttons
* Minimal and clean layout
* Card-based treatment sections
* Clear confirmation feedback

---

## Visual Design

The interface uses:

* white backgrounds for a clean medical feel
* dark typography for readability
* deep red accent colors inspired by the clinic branding

---

## Tools Used

* Figma
* Lucide Icons

---

## Outcome

The final design improves the booking experience by making appointments faster, more organized, and easier to manage for both users and clinic staff.
